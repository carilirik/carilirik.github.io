3. I wanna go for you? apa terjemahannya


i wanna go for you?

aku ingin pergi untuk kamu?

arti perkata :

i : saya / aku

wanna : ingin

go : pergi

for : untuk

you : kamu

I : Saya

Wanna or want to : ingin

Go : pergi

For : demi atau untuk

You : kamu


It should be typed "Saya ingin pergi demi kamu"
