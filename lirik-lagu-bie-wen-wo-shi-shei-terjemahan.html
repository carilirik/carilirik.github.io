9. apa arti zhe shi shei​


Pertanyaan : Arti Dari "Zhè Shì Shéi"

Jawaban : Hanzi : 这是谁?Pinyin : Zhè Shì Shéi?Arti : Ini Siapa?

Penjelasan : 这 = Zhè = Ini是 = Shì = Adalah 谁 = Shéi = Siapa

Contoh Kalimat Dengan 谁 ：她是谁？她是你的女朋友吗？Tā Shì Shéi? Tā Shì Nǐ De Nǔ Péng Yòu Ma?Dia(Perempuan) Siapa? Dia(Perempuan) Pacar Perempuan Kamukah?

Kosakata :她 = Tā = Dia (Perempuan)是 = Shì = Adalah谁 = Shéi = Siapa你 = Nǐ = Kamu 的 = De = Partikel女朋友 = Nǔ Péng Yòu = Pacar Perempuan女 = Nǔ = Perempuan朋友 = Péng Yòu = Teman
