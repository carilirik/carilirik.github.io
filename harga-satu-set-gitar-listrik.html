Jawaban:

LANGKAH PERTAMA (I)

Buatlah persamaan dari pernyataan "Harga buku gambar adalah ¹/₃ harga jangka" yaitu :

x = ¹/₃ y ...(Persamaan 1)

LANGKAH KEDUA (II)

Buatlah persamaan dari pernyataan "Harga jangka adalah ²/₃ harga pastel warna" yaitu :

y = ²/₃ z ...(Persamaan 2)

LANGKAH KETIGA (III)

Hitung nilai z dengan menggunakan persamaan dari pernyataan "Harga 2 pastel warna adalah Rp. 22.500" yaitu :

2z = 22.500

z = \frac{22.500}{2}

z = 11.250

LANGKAH KEEMPAT (IV)

Subitusikan nilai z pada persamaan 2 sehingga diperoleh nilai y dengan menggunakan cara sebagai berikut :

y = ²/₃ z

y = ²/₃ (11.250)

y = 7.500

LANGKAH KELIMA (V)

Subitusikan nilai y pada persamaan 1 sehingga diperoleh nilai y dengan menggunakan cara sebagai berikut :

x = ¹/₃ y

x = ¹/₃ (7.500)

x = 2.500

LANGKAH KEENAM (VI)

Hitung jumlah jangka dengan menggunakan cara sebagai berikut :

8x + ay + 2z = 65.000

8 (2.500) + a (7.500) + 2 (11.250) = 65.000

20.000 + 7.500a + 22.500 = 65.000

42.500 + 7.500a = 65.000

7.500a = 65.000 - 42.500

7.500a = 22.500

a = \frac{22.500}{7.500}

a = 3

∴ Kesimpulan jumlah jangka yang dibeli doni adalah 3 buah.

14. mengapa bentuk gitar listrik berbeda dengan gitar biasa????


Akustik dan Elektrik adalah jenis dari alat musik gitar yang memiliki perannya sendiri dalam penggunaan permainannya, bentuk dan suara ketika dimainkan, serta genrenya. Pada dasarnya akustik dan elektrik sama dalam hal permainannya karena tidak ada perbedaan yang jauh dalam memainkan musiknya, yaitu sama-sama dipetik. Yang paling membedakan antara akustik dan elektrik tentu saja adalah bentuk dan suara yang dihasilkankarena, gitar listrik harus memerlukan daya listrik, sedangkan gitar biasa dapat bersuara lebih besar tanpa daya listrik.

Satu set sofa dijual dengan potongan harga sebesar Rp 450.000,00. Pak Batu membeli satu set sofa tersebut dengan membayar Rp2.100.000,00 dan mendapatkan kembalian Rp50.000,00.

Aritmatika sosial adalah materi matematika yang berhubungan dengan kehidupan sehari", seperti menghitung harga jual, harga beli, untung, rugi, diskon, rabat, bruto, netto, tara, tangungan awal, tabungan ahir, bunga dll.


Pembahasan

Diketahui:

Diskon = Rp 450.000

Membayar = Rp 2.100.000

Kembalian = Rp 50.000

Ditanya :

A. Harga satu set sofa sebelum mendapat potongan harga ?

B. Persentase potongan harga ?

Jawab:

Harga satu set sofa sebelum mendapat potongan harga

= Uang yang diobayar pak datu + diskon - uang kembalian

= Rp 2.100.000 + Rp 450.000 - Rp 50.000

= Rp 2.500.000


Persentase potongan harga

= [tex]\frac{Diskon}{HargaAwal}[/tex] x 100%

= [tex]\frac{450.000}{2.500.000}[/tex] x 100%

= 0,18 x 100%

= 18%

Seorang pedagang menjual satu set meja belajar dengan harga 2.200.000 dan memperoleh keuntungan sebesar 10% maka harga pembelian satu set meja belajar tersebut adalah

______

Persentase Untung:

= 100 + 10

= 110%

Harga beli:

[tex] \frac{100}{110} \times 2.200.000 = 2.000.000[/tex]

Harga pembeliannya adalah Rp2.000.000

Penyelesaian :

Harga jual = Rp2.200.000

Keuntungan = 10%

Maka harga jual Satu set meja belajar :

[tex] = \frac{100\%}{100\% + \%untung} \times harga \: jual \\ = \frac{100\%}{100\% + 10\%} \times 2.200.000 \\ = \frac{100\%}{110\%} \times 2.200.000 \\ = \frac{100}{110} \times 2.200.000 \\ = 100 \times 20.000 \\ = 2.000.000[/tex]

Jadi harga beli satu set meja belajar tersebut adalah Rp2.000.000
